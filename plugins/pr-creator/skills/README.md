# PR Creator Skills

pr-creatorプラグインのスキル定義ディレクトリ

## ディレクトリ構造

```
skills/
├── README.md
└── pr-review-comment/
    └── SKILL.md
```

## スキル一覧

### pr-review-comment（ユーザー呼び出し可能）

PRの変更点を分析し、適切なレビューコメントを作成するスキル。

**トリガー:** 「PRをレビュー」「コードレビュー」「変更点を確認」など

**機能:**
- PRのdiffを取得・分析
- コード品質、バグ、セキュリティ、パフォーマンスの観点でレビュー
- 優先度付きのレビューコメントを生成
- ポジティブフィードバックも含む

**使用例:**

```
# ローカル変更のレビュー
PRの変更点をレビューして

# GitHub PRのレビュー
PR #123 をレビューして
https://github.com/owner/repo/pull/123 をレビューして
```

**レビュー観点:**
- コード品質（可読性、複雑度、DRY原則）
- バグ・エラー（境界値、null/undefined、例外処理）
- セキュリティ（インジェクション、認証・認可）
- パフォーマンス（N+1クエリ、メモリリーク）
- テスト（カバレッジ、エッジケース）

## スキルの追加方法

新しいスキルを追加する場合は、ディレクトリを作成して `SKILL.md` ファイルを配置してください：

```
skills/
└── new-skill-name/
    └── SKILL.md
```
