# PROJECT_REFERENCES.md テンプレート

このファイルをプロジェクトルートまたは `.claude/` ディレクトリに配置してください。
`/onboarding` や `/find-files` コマンドがこのファイルを参照し、
プロジェクト固有の用語に基づいた検索精度を向上させます。

---

## 使い方

1. このテンプレートを `.claude/PROJECT_REFERENCES.md` にコピー
2. 「例:」で始まる行を削除し、実際のプロジェクト情報に置き換え
3. 空の行は削除

```bash
# コピーコマンド
mkdir -p .claude
cp PROJECT_REFERENCES_TEMPLATE.md .claude/PROJECT_REFERENCES.md
```
